# Packet Tracer Labs — CCNA & Networking Fundamentals

## Overview

This repository contains my **Cisco Packet Tracer labs** focused on networking fundamentals, CCNA-level concepts, and practical troubleshooting scenarios. These labs demonstrate hands-on understanding of network design, configuration, and security basics rather than theory only.

The goal of this repository is to show **job-ready networking skills** for roles such as **IT Support, Network Technician, Junior Sysadmin, and NOC roles**, especially in remote or entry-level environments.

---

## Skills Demonstrated

* Network topology design (small-to-medium networks)
* TCP/IP fundamentals
* VLAN configuration & inter-VLAN routing
* Static routing & basic dynamic routing concepts
* DHCP configuration and troubleshooting
* DNS concepts
* Switch & router configuration (CCNA-level)
* Network security basics (port security, ACL concepts)
* Troubleshooting connectivity issues

---

## Tools Used

* **Cisco Packet Tracer**
* Cisco IOS (simulated)

## Lab Structure

Each lab is organized as follows:

```
/lab-name
  ├── README.md        # Lab explanation & objectives
  ├── topology.png     # Network topology diagram (exported)
  └── lab.pkt          # Packet Tracer file
```

## How to Use These Labs

1. Download the `.pkt` file
2. Open it using **Cisco Packet Tracer**
3. Review device configurations
4. Test connectivity using ping and simulation mode
